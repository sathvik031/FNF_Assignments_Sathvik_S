Working with Web Forms

Forms are used to take inputs from the User and process it to a storage location like a Database or a local storage where the inputs are stored in the browser itself. In this case, the data is available to the current machine only.

Forms are usually processed in the server(Web Server).If the App is using client side scripting, then the form shall skip some attributes to avoid posting the data to the server.

All the inputs expected from the User shall be within the form tag. Tney are are basically controls, also called as HTML Controls.
